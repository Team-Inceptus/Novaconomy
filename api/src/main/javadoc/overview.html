Novaconomy, a free to use multi-economy plugin for Bukkit, Spigot, and Paper, supporting 1.8 and above.

The documentation is for integrating with the Novaconomy API, provided by the plugin. This documentation does not cover how to set the Novaconomy API up, how to install it, or setting up a workspace. Working knowledge of the Java language is a prerequisite for developing plugins when integrating with this plugin.
